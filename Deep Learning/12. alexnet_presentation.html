AlexNet Architecture Overview

Total Parameters: 62,378,344

8 Layers: 5 Convolutional + 3 Fully Connected

Key Mathematical Formulas

Convolution Output Size Formula:

Output = (Input - Filter + 2×Padding) / Stride + 1

Max Pooling Output Size Formula:

Output = (Input - Pool_size) / Stride + 1

Parameter Count Formulas:

Convolutional Layers:

Parameters = (Filter_W × Filter_H × Input_Channels + 1) × Num_Filters

Fully Connected Layers:

Parameters = (Input_Size × Output_Size) + Output_Size

Max Pooling Operation:

For a 3×3 pooling window:

Output = max(a, b, c, d, e, f, g, h, i)

Where a through i are the nine values in the pooling window

ReLU Activation Function:

f(x) = max(0, x)

Convolutional Layers Calculations (Part 1)

Conv-1: First Convolutional Layer

Input: 227×227×3, Filters: 96, Size: 11×11, Stride: 4, Padding: 0

Output = (227 - 11 + 0) / 4 + 1 = 216/4 + 1 = 54 + 1 = 55

Result: 55×55×96

Parameters = (11×11×3 + 1) × 96 = (363 + 1) × 96 = 34,944

Conv-2: Second Convolutional Layer

Input: 27×27×96, Filters: 256, Size: 5×5, Stride: 1, Padding: 2

Output = (27 - 5 + 4) / 1 + 1 = 26/1 + 1 = 27

Result: 27×27×256

Parameters = (5×5×96 + 1) × 256 = (2400 + 1) × 256 = 614,656

Conv-3: Third Convolutional Layer

Input: 13×13×256, Filters: 384, Size: 3×3, Stride: 1, Padding: 1

Output = (13 - 3 + 2) / 1 + 1 = 12/1 + 1 = 13

Result: 13×13×384

Parameters = (3×3×256 + 1) × 384 = (2304 + 1) × 384 = 885,120

Convolutional Layers Calculations (Part 2)

Conv-4: Fourth Convolutional Layer

Input: 13×13×384, Filters: 384, Size: 3×3, Stride: 1, Padding: 1

Output = (13 - 3 + 2) / 1 + 1 = 13

Result: 13×13×384

Parameters = (3×3×384 + 1) × 384 = (3456 + 1) × 384 = 1,327,488

Conv-5: Fifth Convolutional Layer

Input: 13×13×384, Filters: 256, Size: 3×3, Stride: 1, Padding: 1

Output = (13 - 3 + 2) / 1 + 1 = 13

Result: 13×13×256

Parameters = (3×3×384 + 1) × 256 = (3456 + 1) × 256 = 884,992

Key Observations:

  • First layer: Large 11×11 filters with stride 4 for aggressive downsampling
  • Later layers: Smaller 3×3 filters maintain spatial resolution
  • Channel growth: 3 → 96 → 256 → 384 → 384 → 256
  • Parameter distribution: Later conv layers have more parameters due to higher channel counts

Pooling Layer Calculations

MaxPool-1: First Max Pooling Layer

Input: 55×55×96, Pool Size: 3×3, Stride: 2, Padding: 0

Output = (55 - 3) / 2 + 1 = 52/2 + 1 = 26 + 1 = 27

Result: 27×27×96

Parameters: 0 (pooling layers have no trainable parameters)

MaxPool-2: Second Max Pooling Layer

Input: 27×27×256, Pool Size: 3×3, Stride: 2, Padding: 0

Output = (27 - 3) / 2 + 1 = 24/2 + 1 = 12 + 1 = 13

Result: 13×13×256

Parameters: 0

MaxPool-3: Third Max Pooling Layer

Input: 13×13×256, Pool Size: 3×3, Stride: 2, Padding: 0

Output = (13 - 3) / 2 + 1 = 10/2 + 1 = 5 + 1 = 6

Result: 6×6×256

Parameters: 0

Max Pooling Benefits:

  • Translation Invariance: Small shifts in input don't affect output
  • Dimensionality Reduction: Reduces computational load for subsequent layers
  • Feature Selection: Retains strongest activations in each region
  • Overfitting Prevention: Acts as regularization by reducing parameters

Fully Connected Layer Calculations

FC-1: First Fully Connected Layer

Input: 6×6×256 = 9,216 units (flattened), Output: 4,096 neurons

Activation: ReLU with Dropout (0.5)

Parameters = (9,216 × 4,096) + 4,096 = 37,748,736 + 4,096 = 37,752,832

Note: This layer contains 60.5% of all network parameters!

FC-2: Second Fully Connected Layer

Input: 4,096 neurons, Output: 4,096 neurons

Activation: ReLU with Dropout (0.5)

Parameters = (4,096 × 4,096) + 4,096 = 16,777,216 + 4,096 = 16,781,312

Note: This layer contains 26.9% of all network parameters

FC-3: Output Layer (Classification)

Input: 4,096 neurons, Output: 1,000 neurons (ImageNet classes)

Activation: Softmax for probability distribution

Parameters = (4,096 × 1,000) + 1,000 = 4,096,000 + 1,000 = 4,097,000

Note: Final classification layer with 6.6% of total parameters

Dropout Mechanism:

During training, randomly sets 50% of neurons to zero to prevent overfitting and improve generalization.

Advantages & Disadvantages

Advantages

  • Breakthrough Performance: Dramatically improved ImageNet accuracy
  • ReLU Innovation: Faster training and deeper networks
  • Dropout Regularization: Effective overfitting prevention
  • GPU Acceleration: Pioneered parallel deep learning
  • Data Augmentation: Increased effective dataset size
  • Scalable Architecture: Demonstrated deep networks work
  • Feature Learning: Automatic hierarchical feature extraction
  • Transfer Learning: Pre-trained features work on other tasks
  • Industry Impact: Sparked the modern AI revolution

Disadvantages

  • High Parameter Count: 62M parameters, prone to overfitting
  • FC Layer Dominance: 87% parameters in FC layers
  • Large Memory Requirements: Heavy computational resources
  • Fixed Input Size: Requires 227×227 input images
  • No Batch Normalization: Training instability issues
  • Limited Depth: Only 8 layers vs modern standards
  • Manual Feature Engineering: Some preprocessing still required
  • Vanishing Gradients: Still challenging for very deep networks
  • Outdated by Modern Standards: Superseded by ResNet, EfficientNet, etc.

AlexNet's Legacy & Impact

  • Training Techniques: Dropout, data augmentation still essential
  • Scaling Laws: More data + bigger models = better performance
  • GPU Acceleration: Parallel computing is now standard
